To install:
 -	Install the following on the machine:
 	- zsh
	- neovim/vim
	- tmux
 - git clone --recursive https://github.com/ccheek21/dotfiles.git ~/.dotfiles
 - chsh -s /bin/zsh
 - ~/.dotfiles/install
 - run 'zplug install'
 - follow instructions to install deocomplete-clang
 - in vim, run ':PlugInstall'
